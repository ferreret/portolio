## 7. Riesgos identificados

| Riesgo | Probabilidad | Impacto | Mitigación |
|---|---|---|---|
| `ghchart.rshah.org` deja de funcionar (servicio de terceros gratuito) | Media | Bajo | Fallback: la sección muestra solo top languages (API GitHub directa). Alternativa: self-host del generador de charts |
| API pública de GitHub alcanza rate limit (60 req/h sin auth) | Baja | Bajo | El componente maneja el error silenciosamente — la sección de languages simplemente no aparece |
| Bundle JS crece 39 kB por react-router-dom | — | Bajo | Aceptado — el beneficio en mantenibilidad y UX (deep links, back/forward nativo) lo justifica |
| Umami no trackea cambios de ruta client-side tras migrar a React Router | Media | Medio | Verificar en dashboard de Umami; si falla, añadir `useEffect` con `umami.track()` en `ScrollToTop` |
| Publicación accidental de información confidencial de clientes Tecnomedia | Baja | **Alto** | Regla en memoria persistente + revisión manual obligatoria |
